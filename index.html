RubyTree by evolve75 / anupamsg

A General Purpose Tree Data Structure for Ruby.

RubyTree is a simple and general purpose implementation of the Tree data structure for the Ruby language.

RubyTree aims to provide a simple to use API model for the conceptual tree, and the ability to extend the model into specialized cases in user-generated code.

The current version provides implementations of:

  • A general purpose tree structure, and
  • A binary tree

Future versions may include other tree implementations such as the Red-Black tree.

RubyTree is available as a RubyGem and is released under the BSD License.

RubyTree is hosted at http://rubytree.rubyforge.org

Documentation

Documentation is available for the current released RubyGem at:
http://rubytree.rubyforge.org/rdoc/
Documentation for the current Git HEAD version is available at:
http://rdoc.info/projects/evolve75/RubyTree

Dependencies

  • Build Dependency on Hoe for generating the Gemspec
  • Soft dependency on the structured_warnings RubyGem (optional)

Install

$ gem install rubytree

You may need to run this as a super-user on your system (root or sudo).
